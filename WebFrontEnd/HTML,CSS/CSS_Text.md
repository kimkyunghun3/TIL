```
text-align
문자 정렬 방식을 지정

left 왼쪽 정렬
right   오른쪽 정렬
center  가운데 정렬
justify 양쪽 맞춤 ( 2줄 이상일 경우 적용 된다)

text-decoration
문자의 장식을 설정한다
선에 대한 장식을 설정하는 것이다.

none 선 없음
uderline 밑줄을 지정
overline 윗줄을 지정
line-throught 중앙선을 지정

text-indent
(첫 번째줄) 들여쓰기 지정
음수 값을 사용할 수 있으며 음수 값을 사용하면 첫째 줄은 왼쪽으로 들여쓰기 된다.

letter-spacing
글자간의 자간을 설정할 수 있다.
noraml 단어 사이의 일반 간격
px, em ,cm 으로 설정가능하다.

word-spacing 
단어 사이의 간격을 설정할 수 있다.(띄어쓰기)
letter-spacing 이랑 설정은 같다.
normal 은 0을 말한다. 이는 일반 간격을 말하는 것이다.
```
